# Restaurant Pizza Management System

Welcome to the Restaurant Pizza Management System, a user-friendly web application built with Flask. This system simplifies restaurant and pizza management, providing an intuitive interface for adding, viewing, and deleting restaurants, pizzas, and their associations. Perfect for restaurant owners and managers, this tool streamlines menu customization and organization.

## Features

### Restaurant Management:

- **Add Restaurants:** Easily add new restaurants with unique names and addresses.
- **View Restaurants:** Access a list of existing restaurants for quick reference.
- **Delete Restaurants:** Remove unwanted restaurants effortlessly.

### Pizza Management:

- **Add Pizzas:** Intuitively add new pizza types with names, ingredients, and prices.
- **View Pizzas:** Explore a comprehensive list of existing pizzas.
- **Delete Pizzas:** Remove outdated or unused pizza types with a single click.

### Restaurant-Pizza Associations:

- **Create Associations:** Seamlessly link restaurants with specific pizzas, complete with customizable prices.
- **View Associations:** Access a detailed overview of existing restaurant-pizza connections.
- **Delete Associations:** Easily manage menu offerings by removing specific associations.

## Getting Started

### Prerequisites

Before you begin, ensure you have the following requirements:

- **Python:** Version 3.6 or higher
- **Pip:** Python package manager
- **Git:** (Optional) Version control system

### Installation

1. **Clone the Repository:**
   ```
   git clone <repository-url>
   ```

2. **Navigate to the Project Directory:**
   ```
   cd restaurant-pizza-management
   ```

3. **Install Dependencies:**
   ```
   pip install -r requirements.txt
   ```

4. **Run the Application:**
   ```
   python app.py
   ```

5. **Access the Web Application:**
   Open a web browser and navigate to http://localhost:5555.

## Usage

### Restaurant Management:

1. **View Restaurants:**
   - Click on the "Restaurants" tab to view the list of restaurants.
  
2. **Add a New Restaurant:**
   - Click "Add Restaurant" to create a new restaurant.
  
3. **Manage Restaurants:**
   - Click on a restaurant to view its details or delete it.

### Pizza Management:

1. **View Pizzas:**
   - Click on the "Pizzas" tab to view the list of pizza types.
  
2. **Add a New Pizza:**
   - Click "Add Pizza" to create a new pizza type.
  
3. **Manage Pizzas:**
   - Click on a pizza to view its details or delete it.

### Restaurant-Pizza Associations:

1. **View Associations:**
   - Click on the "Restaurant Pizzas" tab to view existing associations.

2. **Create a New Association:**
   - Click "Add Association" to create a new association.

3. **Manage Associations:**
   - Click on an association to view its details or delete it.

## Database Schema

The application uses the following database schema:

- **restaurants:** Stores restaurant information.
- **pizzas:** Stores pizza information.
- **restaurant_pizzas:** Stores associations between restaurants and pizzas, including prices.
